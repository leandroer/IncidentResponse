# Security Policy

## Reporting a vulnerability

Please use [GitHub private vulnerability reporting](https://github.com/leandroer/IncidentResponse/security/advisories/new). Do not disclose an unremediated vulnerability in a public issue.

Include the affected page or file, reproduction steps, observed impact, and any suggested mitigation. Do not include credentials, personal information, customer data, or unrelated system details.

This repository is a static educational site. Reports about factual corrections, broken links, or editorial issues can use a standard GitHub issue; reserve private reporting for security weaknesses.
